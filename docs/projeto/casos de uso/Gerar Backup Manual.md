
## Atores
- Professor

## Pré-condição
- Sistema possui dados locais armazenados.

## Fluxo Principal
- Professor acessa Configurações.
- Seleciona Gerar Backup.
- Sistema compacta o banco local (SYS-02).
- Usuário salva o arquivo.

## Pós-condição
- Arquivo de backup gerado.

## Fluxos Alternativos
-Fazer backup sem salvar as alterações, retornando um backup falso.
