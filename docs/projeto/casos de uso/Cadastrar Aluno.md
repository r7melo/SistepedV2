## Atores
- Professor

## Pré-condição
- Professor tem acesso ao menu de alunos.

## Fluxo Principal
- Professor acessa o menu Alunos.
- Seleciona Adicionar Aluno.
- Preenche nome, idade, turma e contato.
- Confirma o cadastro.
- Sistema salva no banco local.

## Pós-condição
- Aluno cadastrado no sistema.

## Fluxos Alternativos
- Realizar o backup manual.
